Markdown 语法介绍

1.1 标题

格式：
  # 一级标题
  ## 二级标题
  ### 三级标题
  #### 四级标题
  ##### 五级标题
  ###### 六级标题

快捷键：
  通过左右中括号[]来控制标题级别ctrl + shift + ]

1.2 字体效果

格式：
  用一个 * 包含一段文本就是斜体的语法
  用 ** 包含一段文本就是粗体的语法
  用 *** 包含一段文本就是加粗斜体的语法
  用 ~~ 包含一段文本就是删除线的语法

效果：
  我是 粗体  ctrl + B
  我是 斜体  ctrl + I
  我是 加粗斜体   ctrl + I B
  我是 删除线

1.8 表格

  1. 不管是哪种方式，第一行为表头，第二行分隔表头和主体部分，第三行开始每一行为一个表格行。
  2. 列于列之间用管道符|隔开。原生方式的表格每一行的两边也要有管道符。
  3. 第二行还可以为不同的列指定对齐方向。默认为左对齐，在-右边加上:就右对齐。
